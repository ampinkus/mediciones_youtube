Module: app

Punto de entrada principal de la aplicación Express. Configura middlewares, motor de plantillas y rutas. Este módulo sirve como bootstrap de todo el sistema: - Carga rutas principales del sistema. - Configura el motor de vistas con EJS. - Sirve archivos estáticos como scripts y estilos desde /public y /modulos.
